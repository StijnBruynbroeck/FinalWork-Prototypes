UNITY SETUP INSTRUCTIES VOOR API CALL SCRIPT

1. SCENE SETUP:
   - Maak een leeg GameObject en noem het "ApiManager"
   - Voeg het ApiCall.cs script toe aan de ApiManager
   - Zorg ervoor dat de ApiManager in je scene aanwezig is

2. UI ELEMENTEN NODIG:
   - Maak twee TextMeshPro - Text elementen:
     * Eén voor resultText (toont transcriptie resultaten)
     * Eén voor statusText (toont status updates)
   - Sleep deze elementen naar de corresponderende velden in de ApiCall component

3. MICROFOON PERMISSIES:
   - In Unity Edit > Project Settings > Player:
     * Bij "Other Settings", zet "Microphone Usage Description" aan
     * Voeg een beschrijving toe (bv: "Microfoon nodig voor spraakherkenning")
   - Voor iOS: Zet "Require Microphone" aan bij iOS settings
   - Voor Android: Voeg toe aan AndroidManifest.xml:
     <uses-permission android:name="android.permission.RECORD_AUDIO" />

4. NETWERK PERMISSIES:
   - Voor iOS: Zet "Allow HTTP to Any Domain" aan OF gebruik HTTPS URLs
   - Voor Android: Voeg toe aan AndroidManifest.xml:
     <uses-permission android:name="android.permission.INTERNET" />

5. PACKAGES NODIG:
   - TextMeshPro package (standaard in moderne Unity versies)
   - Unity Web Request (standaard aanwezig)

6. API INSTELLINGEN:
   - De API key staat al in het script: "eb3b47c9928a4faba73a6a2500b97bd5"
   - Base URL is ingesteld op: "https://api.assemblyai.com"
   - Test of deze key nog geldig is

7. TESTEN:
   - Start de scene
   - Druk op T toets en houd vast om op te nemen
   - Laat T los om transcriptie te starten
   - Bekijk resultaten in de UI elementen

8. EVENTUELE PROBLEMEN:
   - Als microfoon niet werkt: check permissies in Player Settings
   - Als API calls falen: check internet connection en API key
   - Als UI niet werkt: zorg dat TextMeshPro components correct gelinkt zijn

9. EXTRA FEATURES:
   - Script heeft ook UploadAudioForTranscription() methode
   - Kan gebruikt worden voor audio bestanden via code
   - Recorder settings zijn aanpasbaar in Inspector

10. BUILD INSTELLINGEN:
     - Zorg dat juiste platform geselecteerd is (iOS/Android/PC)
     - Check permissies voor mobiele platforms
     - Test op target apparaat voor microfoon functionaliteit

MICROCONTROLLER SCRIPT MANUAL
---

11. MICCONTROLLER SETUP:
    - Maak een leeg GameObject en noem het "MicController"
    - Voeg een AudioSource component toe aan hetzelfde GameObject
    - Voeg het MicController.cs script toe aan het GameObject
    - Het script automatisch een AudioSource component vereisen

12. FUNCTIONALITEIT:
    - Automatisch detecteren en starten van de eerste beschikbare microfoon
    - Real-time volume monitoring van de microfoon input
    - Continue audio capture met loop (10 seconden buffer, 44100 Hz)
    - Automatic cleanup wanneer het script wordt disabled

13. WERKING:
    - Start(): Detecteert microfoon en start opname automatisch
    - Update(): Monitor microfoon volume elke frame
    - StartMicrophone(): Start continue opname met AudioSource
    - OnDisable(): Stopt opname netjes wanneer component wordt uitgeschakeld

14. VOLUME MONITORING:
    - Leest elke frame 128 samples van de microfoon
    - Berekent gemiddelde absolute amplitude
    - Logt volume naar console wanneer niveau > 0.01f (1%)
    - Volume wordt weergegeven als percentage (level * 100)

15. AUDIO INSTELLINGEN:
    - Opname formaat: 10 seconden loop buffer
    - Sample rate: 44100 Hz
    - Continuous recording: true (niet stoppen na 10 seconden)
    - Loop playback: true

16. GEBRUIK VOORBEELDEN:
    - Volume indicator voor spraakherkenning
    - Audio level visualization
    - Detectie wanneer er wordt gesproken
    - Real-time audio monitoring

17. DEBUG INFORMATIE:
    - Logt geselecteerde microfoon: "Geselecteerde microfoon: [naam]"
    - Logt volume wanneer > 1%: "Microfoon Volume: [percentage]"
    - Logt fout als geen microfoon: "Geen microfoon gevonden!"
    - Logt stop bericht: "Microfoon opname gestopt."

18. INTEGRATIE TIPS:
    - Combineer met ApiCall script voor spraak-naar-tekst
    - Gebruik volume data voor visualisaties
    - Voeg threshold detectie toe voor activiteit detectie
    - Kan uitgebreid worden met audio processing

19. TROUBLESHOOTING:
    - Geen audio: Check of AudioSource component aanwezig is
    - Geen microfoon: Check microfoon permissies in Player Settings
    - Geen logs: Check Unity Console voor foutmeldingen
    - Performance: Reduce sample buffer indien nodig

20. PLATFORM SPECIFIEK:
    - Web: Werkt alleen in HTTPS omgeving met microfoon permissie
    - Mobile: Vereist runtime microfoon permissie van gebruiker
    - Standalone: Werkt met systeem microfoon apparaten
    - Editor: Werkt met development microfoon setup
